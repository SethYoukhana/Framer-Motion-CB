Intro to Framer Motion

What is Framer Motion?

An animation library used with:
Vanilla Javascript
React
Vue

What can be animated?

  • CSS properties (like opacity, transform and filter)
  • SVG attributes and paths
  • Independent transforms (x, rotateY etc)
  • JavaScript objects (containing strings/colors/numbers)

Set up/Installation

Quickstart guide
1. "npm install motion" in terminal
2. Place script tag in header
3. Animate away!
